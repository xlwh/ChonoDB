### 1. 查询功能限制
- 查询引擎的部分测试失败，可能是因为 `MemStore` 的查询实现不完整
- 标签索引功能需要进一步完善
- 查询缓存的集成需要优化

### 2. 分布式功能限制
- 分布式集成测试已创建但主要测试配置和分片功能
- 完整的集群功能测试需要多节点环境
- 数据复制和故障转移需要更多测试

### 3. 存储后端限制
- GCS 测试需要有效的 Google Cloud 凭证
- S3 和 MinIO 的集成测试需要配置

### 4. CLI 工具限制
- CLI 工具的部分功能依赖于完整的 `MemStore` API
- 一些高级功能（如标签值查询）需要完善
